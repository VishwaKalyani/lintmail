## 📖 What is lintmail?

lintmail is a Chrome extension designed to check your HTML emails. It helps you find potential issues before you send your emails to customers or coworkers. The tool reviews color contrast to meet accessibility standards. It detects which email service provider (ESP) you use. lintmail checks if your email will work in popular email clients like Outlook and Gmail. It also helps you test different versions of your emails for better results.  

This extension is aimed at people who create emails but do not have technical skills. You do not need to know coding to use it. lintmail works inside your Chrome browser, making it easy to access while you work.

---

## ⚙️ Features

- **Contrast audit**: Ensures your email text and background have enough contrast to be readable for everyone.
- **ESP detection**: Identifies the email service provider behind your message.
- **Client compatibility report**: Finds issues that can cause emails to display wrong in email apps.
- **A/B testing support**: Helps you prepare different email versions and test which works best.
- **Side panel integration**: Opens next to your Gmail or web email editor for easy access.
- **Manifest V3 support**: Uses the latest Chrome extension format for safety and speed.

## 📂 How lintmail works

lintmail scans the HTML code of your email. It checks for problems like:

- Text with low contrast against background colors.
- Unsupported HTML and CSS that may break in some email clients.
- Identifies the ESP to help you tailor emails specifically.
- Highlights sections where A/B testing differences appear.

The report updates live. You do not need to reload the page.

## 🛠 Troubleshooting

- If lintmail panel does not appear, check the extension is enabled in Chrome.
- Make sure your browser is up to date.
- Disable other Chrome extensions that may block lintmail.
- If auditing does not start, refresh the email page or close and reopen the side panel.
- For ESP detection errors, confirm you are working in a supported email environment like Gmail or Outlook on the web.
- If you see errors in Chrome, retry installation steps.
